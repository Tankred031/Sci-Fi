# Machines of Future

Retro-futuristic web project inspired by iconic dystopian sci-fi movies of the 1980s.

This project explores the dark technological visions presented in legendary films such as:

* Blade Runner
* The Terminator
* RoboCop

The website was created as a thematic HTML/CSS project focused on:

* atmosphere
* cinematic presentation
* visual styling
* interactive hover effects
* retro cyberpunk aesthetics

---

## Technologies Used

* HTML5
* CSS3
* Flexbox Layout
* Custom Hover Effects
* Responsive Design Basics

---

## Features

* Multi-page structure
* Movie-themed navigation
* Interactive poster glow effects
* Custom visual themes for each movie
* Audio and media integration
* Retro-futuristic design approach

---

## Project Goal

The goal of this project was to practice frontend fundamentals while creating an immersive cinematic experience inspired by classic sci-fi dystopias.

Rather than building a traditional business-style website, the project focuses on atmosphere, presentation, and thematic storytelling through web technologies.
